Welcome

This is some welcome text to give a brief description of the site.

This theme originally came in multiple variations that included a full width version, one with a sidebar and one with multiple content segments. This version combines all those features into one using multiple, variable-width, columns. Download the stylesheet.

By using percentages, rather than fixed pixel lengths, page columns may be created that expand to the available container width without crowding the page segment. The theme uses only two images (besides the icons that are part of the content): The magnifying glass used as the search button in a lot of my templates and the background pattern. For more information on the theme, please see the accompanying blog post
